# Reports

This directory contains AI-generated reports, audits, and analytical artifacts.

## Purpose

Reports are **temporary analytical artifacts** created during development.

They should:
- Provide audit trails and analysis snapshots
- Document cleanup efforts and verification results
- Capture point-in-time assessments
- Support decision-making and project governance

They should NOT:
- Be treated as canonical documentation
- Replace permanent docs in `/docs`
- Be referenced as authoritative governance
- Contain long-term specifications or standards

## Lifecycle

Reports may be archived or removed after they are no longer relevant. For permanent governance rules and standards, reference documentation in `/docs`.

## Current Reports

Reports will be listed here as they are created during the project lifecycle.
