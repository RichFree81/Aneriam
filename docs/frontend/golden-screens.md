# Golden Reference Screens (MUI-F4)

## Purpose
These pages serve as a "live specification" for our standards. 

> [!WARNING]
> **Status: Specification Only**. These routes are defined here as standards but are **NOT YET IMPLEMENTED** in the codebase. Do not attempt to visit them.

## Registry

### 1. Forms Standard (`/forms-test`) - *To Be Implemented*
- **Milestone**: C
- **Validates**:
  - Input density and spacing.
  - Validation error states (field-level and global).
  - Form layout (grid alignment).
  - Button placement (Submit/Cancel).

### 2. Feedback Standard (`/feedback-test`) - *To Be Implemented*
- **Milestone**: D
- **Validates**:
  - Dialogs (Confirm, Alert).
  - Notifications (`Snackbar`).
  - Empty / Error / Loading states.

### 3. Data Standard (`/data-test`) - *To Be Implemented*
- **Milestone**: E
- **Validates**:
  - `DataToolbar` layout and wrapping.
  - Data formatting (dates, currency).
  - Pagination controls.
  - Empty/Error states within data contexts.

## Usage
When implemented, these will be available as DEV-ONLY routes.
CI/CD integration: Future automation may screenshot these routes.
